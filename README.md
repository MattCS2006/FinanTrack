# FinanTrack

Proyecto para desarrollar una app móvil que ayuda a gestionar finanzas personales.

## Tecnologías

- React Native + Expo
- Supabase (PostgreSQL)
- Prisma ORM
- OpenAI API para asistente inteligente (MVP)
